VEDA BANK - 차희웅 이재용 조

(은행의 간단한 조회 및 입출력을 위한 프로그램)

우선 실행전에 file open을 위한 파일 주소를 써주시면 감사하겠습니다.
ex)"C:/Users/DREAM06/project/account.txt"

<실행 방법>
1. 실행시 로그인과 회원가입의 선택지가 생성됩니다.
2. 버튼을 누르셔서 실행하시고, 계좌가 없으시다면 생성해주시면 되겠습니다.
3. 이후 로그인 후 입출금과 조회의 창이 뜹니다.
4. 조회는 현재 잔액, 입출금은 입출금을 위한 창이 생성됩니다.
5. 입출금을 원하시면 원하는 금액을 적으시고, 자신의 비밀번호를 입력하셔야 진행이 됩니다.
6. 각 창의 종료는 X표시를 누르셔서 종료해주시면 되겠습니다.
7. 이후 파일 입출력을 통해 금액이 저장되어있습니다.
